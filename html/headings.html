  3. style: we use style attribute to specify inline CSS styles for an element.
  4. title: we use title attribute to provide additional information about an element.
  5. href: we use href attribute to specify the URL of a link.
  6. src: we use src attribute to specify the source of an image or video.
  7. alt: we use alt attribute to provide alternative text for an image.
  8. target: we use target attribute to specify where to open the linked document.
